# A simple todo app.

This is lab project for learning about
- Vue
- Vuex
- Directive Sortable
- Vuex Subscribe LocalStorage
- Call Parent Method with $emit

# Screenshot
![Todo](http://i.imgur.com/j1sA4ku.png)

# Features
- Add todo
- Edit todo
- Move todo
- View JSON todo
- Count todo
- Clear todos
- Notification on add
- Delete with confirm

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# run unit tests
npm run unit

# run e2e tests
npm run e2e

# run all tests
npm test
```
